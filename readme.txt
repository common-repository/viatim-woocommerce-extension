=== ViaTim WooCommerce Plugin ===
Tags: woocommerce, viatim
Requires at least: 3.5.1 & WooCommerce 2.0+
Tested up to: 4.8
Stable tag: 2.3.3
Contributors: Roy Sijnesael for ViaTim

Use the ViaTim WooCoommerce Plugin to add the ViaTim delivery option to your store, and add rates for sending packages with ViaTim.

== Description ==

This WooCommerce extension allows you to add your own rates for sending with ViaTim.

= Main features =
- Add delivery rates to your webshop that can be used for sending with ViaTim.

== Select Delivery Options ==
- Select the desired delivery options you want to use with your webshop.

== Same Day Delivery ==
- Add a rate for Same Day delivery.

== Next Day Delivery ==
- Add a rate for Next (standard) Day delivery.

== International Delivery ==
- Add a rate for International delivery.

== Free Delivery ==
- Add a target price for when shipping is supposed to be free.

== Installation ==

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install of ViaTim WooCommerce Plugin, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "ViaTim WooCommerce Plugin" and click Search Plugins. You can install it by simply clicking Install Now. After clicking that link you will be asked if you're sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation via the WordPress interface =
1. Download the plugin zip file to your computer
2. Go to the WordPress admin panel menu Plugins > Add New
3. Choose upload
4. Upload the plugin zip file, the plugin will now be installed
5. After installation has finished, click the 'activate plugin' link

= Manual installation via FTP =
1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

= Setting up the plugin =
1. Go to the menu `WooCommerce > Settings`.
2. Click on the 'Shipping' tab and click on the ViaTim option.
3. Fill in your API key. If you don't have an API key, contact us at it@viatim.nl and we'll make sure you get one.
4. Select your preffered delivery options.
4. Fill in your rates and times.
5. Click 'Save changes'.
6. The plugin is ready to be used!

= Testing =
We advise you to test the whole checkout procedure once to see if everything works as it should.

== FAQ ==

= How do I get an API key for this plugin? =
Send us an e-mail at it@viatim.nl, and we will create an account for our control panel for you. We'll send an API key with these account details.
